Your biggest interview risk is coding atrophy, not lack of AI knowledge. Fix this first — everything else depends on it.
Critical

2 problems/day on LeetCode — medium first, hard after week 3

DB deep dive: indexing, query plans, ACID, isolation levels
Kafka fundamentals: partitions, consumer groups, exactly-once semantics

Phase 2 — Build + system design (months 2–4)
Build your flagship project AND strengthen system design in parallel. These compound each other.
High impact
Flagship project: config-driven orchestration
Core: rule/workflow engine with YAML/JSON-driven pipeline config
AI layer: LLM integration for decision explanations + anomaly flagging
RAG component: embed loan policy docs, query against them
Observability: structured logs, metrics, tracing (Prometheus + Grafana)
Ship it deployed — a live URL beats a GitHub repo every time
Write one detailed engineering blog post about its architecture
System design mastery
HLD: URL shortener, notification system, rate limiter, Uber-like dispatch, payment ledger
LLD: design a workflow engine, parking lot, LRU cache, pub/sub
Read: Designing Data-Intensive Applications (Kleppmann) — chapters 5, 6, 9, 11
Practice: 2 HLD mocks/week from month 3 onward
AI system design: design a RAG pipeline, LLM gateway, vector search at scale
Project trap: "planning to integrate AI capabilities" is not the same as doing it. Set a hard scope by end of week 2 of Phase 2 and don't expand it. A working narrow system > an ambitious half-built one.
The orchestration platform is perfect for your profile — it directly maps to your LOS experience, shows architectural thinking, and gives concrete AI integration stories. It will carry your interviews.

Phase 4 — Offer and negotiation (month 5–6)
Most engineers leave 5–15 LPA on the table by accepting first offers. Don't be that person.
High leverage
Negotiation playbook
Never reveal your current salary first — it anchors you at 37 LPA
State a target range: "I'm looking at 60–70 LPA based on market"
Always negotiate, even if you want to accept — ask once, clearly
Use competing offers explicitly: "I have another offer at X, can you match?"
Negotiate total comp: fixed + variable + ESOPs + joining bonus separately
Offer evaluation checklist
Fixed component target: 45–55 LPA fixed minimum
ESOP vesting schedule: 4-year cliff vs monthly — matters a lot
Variable/bonus structure: guaranteed vs performance-linked
Growth trajectory: SDE-2 now → SDE-3 in 18 months is better than senior SDE-3 with no headroom
Joining bonus if notice period > 30 days
Your solutioning + cross-functional experience is a premium signal for SDE-3 consideration at some companies. Frame it as "technical leadership without a formal title" — not as reduced coding. This can push you from SDE-2 to SDE-3 band in offers.
