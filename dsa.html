Daily method — non-negotiable
1
Attempt independently for 25 min. No hints, no solutions, no peeking at discuss tab.
2
If stuck at 25 min: read approach only (not code). Close it. Attempt again from scratch.
3
After solving: read the optimal solution even if yours passed. Ask — why is theirs better?
4
Log every problem: date, time taken, hints used (0/1/2), pattern identified, one-line note.
5
Never copy-paste solutions. Type every line. Muscle memory matters in live interviews.
6
Spaced repetition: re-solve problems where you needed hints after 3 days then 7 days. Mark mastered only when solved cleanly twice.
Hard rules
Do not skip order
NeetCode 150 is sequenced intentionally. Arrays before trees. Trees before graphs. Graphs before DP. Jumping ahead causes pattern confusion.
One resource only
NeetCode 150 + NeetCode video explanations. No switching to AlgoExpert, Striver, or LeetCode Explore. Resource switching is procrastination.
Move phases by skill, not calendar
If mediums aren't at 70% by week 8, extend Phase 1. Do not advance because the date says so.
DP scope is limited
Do only the 15 DP problems listed. Advanced DP has poor ROI for your target companies. Resist going deeper.
